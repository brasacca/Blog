Basic JavaScript to Make Static Website Interactive

JavaScript can be used along with HTML and CSS to make a website interactive. JavaScript can be used for form validation, manipulation of CSS styles and many other things.

Firtly, here is an example of how JavaScript can be used to suprise someone with a hidden message! In this example, when the user presses the button on screen, a special message will pop up telling them they are going on a cruise!

Here is the HTML for this webpage, it has a <script> tag which links to the JavaScript file. In the body, there is a button that says "click here" that has a special attribute "onclick" with the value of a JavaScript method that will execute when the button is clicked. There is also a <p> tag with the id "hiddenMessage" and the style attribute making the text hidden.

Now, let's look at the JavaScript file. There is a function revealMessage(). It accesses the HTML file with the variable document and uses the function getElementById which will search for the HTML element with the ID "hiddenMessage". And then it changes that element's style to display='block' which will override the initial value and will make the hidden image visible.
